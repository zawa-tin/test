## 概要

Sparse Tableで区間最大値クエリを処理したい時に、`zawa::sparse_table<structure>`の`structure`に入れる構造体です。

## 機能

`zawa::max_structure<dat_type>`
- `dat_type`
	- `int`や`long long`など、`std::max`に突っ込むことができる型
- **単体テストをまだしていません**
